# Contributing to MemVault

Thanks for your interest in contributing! Here's how to get started:

## Development Setup

```bash
git clone https://github.com/wjy9902/memvault.git
cd memvault
bash scripts/setup.sh
```

## Code Style

- Python: Follow PEP 8
- Shell scripts: ShellCheck clean
- Docstrings for public APIs

## Pull Request Process

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/awesome-feature`)
3. Commit changes (`git commit -m 'Add awesome feature'`)
4. Push to your fork (`git push origin feature/awesome-feature`)
5. Open a Pull Request

## Ideas for Contributions

- SQLite backend option
- Memory export/import (JSON, Markdown)
- Web dashboard
- Alternative decay strategies
- Memory consolidation logic
- Integration examples (LangChain, AutoGen, etc.)
